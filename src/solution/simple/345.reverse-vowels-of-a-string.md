# 345. Reverse Vowels Of A String

[leetcode](https://leetcode-cn.com/problems/reverse-vowels-of-a-string/)

## Solution

### 双指针

分析题目,发现就是元音字母前后交换, 使用前后双指针遍历
需要注意循环的条件

```ts
var reverseVowels = function (s) {
  let start = 0
  let end = s.length - 1
  let startText = ''
  let endReveredText = ''
  const list = ['a', 'e', 'i', 'o', 'u', 'A', 'E', 'I', 'O', 'U']
  while (start <= end) {
    const c1 = s[start]
    const c2 = s[end]
    if (list.includes(c1)) {
      if (list.includes(c2)) {
        startText += c2
        if (start !== end) {
          endReveredText += c1
        }
        start++
      } else {
        endReveredText += c2
      }
      end--
      continue
    }
    startText += c1
    start++
  }

  let endText = ''
  for (let i = endReveredText.length-1; i >= 0; i--) {
    endText += endReveredText[i]
  }
  return startText + endText
};

```
